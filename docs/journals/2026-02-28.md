## Summary

v0.4.0 shipped with TypeScript SDK (agentlens-observe@0.1.0 on npm), PostgreSQL backend support, updated docs, and Docker tag bumped. All code, docs, packages published within ~30 hours of initial feature commits.

## The Brutal Truth

We're finally past single-language SDK hell. Python-only was always a trap — Node.js ecosystem devs couldn't touch AgentLens, so market reach was fundamentally bottlenecked. TS SDK shipped with **zero prod dependencies** using native AsyncLocalStorage, which is elegant but also means we bet hard on Node 18+ being the floor. If that assumption breaks, we're stuck.

Shipping docs update at the same time was necessary but felt rushed — had to cram TS SDK coverage into existing architecture docs without major restructuring. The changelog got long (85 lines added), and there's real integration debt between Python/TS SDKs now.

## What We Tried

1. Started with Python-only SDK — worked great for Python teams, but shut out Node.js market entirely
2. Considered gRPC/protobuf first for cross-language support — too heavyweight for observability, killed that idea
3. Built TS SDK without external deps — AsyncLocalStorage + native fetch is Node's strength; paid off

## Root Cause Analysis (Why This Matters)

**Market Limitation**: Python devs represent ~40% of AI/agent builder mindshare. Node/TS is another 35%. By shipping Python-only, we ignored 35% of addressable market. TS SDK changes that.

**Integration Debt**: We now maintain parallel SDKs. Both need version sync, parity testing, feature parity. Every new feature requires 2x work. No build-once-deploy-everywhere luxury. This is expensive.

**Deployment Surface**: Docker tag needs sync with both SDK versions. Currently manual. Will break eventually when someone publishes TS SDK v0.2 but forgets to update Docker tag.

## Lessons Learned

1. **Multi-SDK Parity = Operational Tax**: Every feature now costs 2x. Need automated parity tests (same test suite runs on both SDKs) before shipping TS v1.0.

2. **Zero Deps Was Right Call**: AsyncLocalStorage + fetch eliminates npm Hell. TS SDK is 11KB; if we used axios + other stuff, would be 150KB+. Kept us nimble.

3. **Docs Lag Real Code**: Updated docs 30 hours AFTER code shipped. Should have documented in parallel. Created 1-2 day window where docs were wrong.

4. **Version Sync Will Bite Us**: PyPI = 0.3.0, npm = 0.1.0, Docker = 0.4.0. Looks intentional, but it's confusing. Next time, all three release at same semver.
